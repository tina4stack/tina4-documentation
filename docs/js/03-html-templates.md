# Chapter 3: HTML Templates

This chapter covers the tina4-js `html` tagged template — the DOM rendering engine.

For the full chapter content, see the [tina4-js book](https://github.com/tina4stack/tina4-book/blob/main/book-5-javascript/chapters/03-html-templates.md) or visit [tina4.com/js](https://tina4.com/js/03-html-templates).

The dev server has rendering issues with \`\${}\` template literal syntax in code examples. The production build at tina4.com renders correctly.
