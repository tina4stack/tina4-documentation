# Introduction

Installation

MacOS / Linux

```bash
python3 -m venv venv
source ./venv/bin/activate
pip install uv
uv sync

```
Windows

```bash  
python -m venv .venv
.venv\Scripts\activate
pip install uv
uv sync
```


Running
```bash
uv run mkdocs serve

```
Building
```bash
uv run mkdocs build

```
