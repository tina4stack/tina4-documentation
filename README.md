# Introduction

Installation

MacOS / Linux

```bash
python3 -m venv venv
source ./venv/bin/activate
pip install mkdocs
pip install --upgrade pip
```
Windows

```bash
python -m venv venv
```


Running
```bash
poetry run mkdocs serve
```
Building
```bash
poetry run mkdocs build
```
